# Dashboard Visualisasi Data
Aplikasi ini dibuat menggunakan **Streamlit** untuk menampilkan visualisasi data penjualan tahun 2022.

**Link Aplikasi**
https://dashboard-visualisasi-data-elsy.streamlit.app/

## Fitur:
- KPI Metrics (Total Sales, Net Profit, AOV, Unique Customers)
- Grafik Penjualan per Kategori
- Tren Penjualan Bulanan
- Heatmap Pendapatan per Bulan & Kategori

## Teknologi:
- Python
- Pandas
- Plotly
- Streamlit

## Jalankan secara lokal:
```bash
streamlit run app.py
